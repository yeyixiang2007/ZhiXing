# 数据持久化说明

## 1. 数据文件位置
- `data/vertex.json`：地点数据
- `data/edge.json`：道路数据
- `data/user.json`：用户数据
- `data/backup/<备份名>/`：备份目录

## 2. 文件格式

### 2.1 vertex.json
```json
[
  {"id":"GATE_E","name":"East Gate","type":"GATE","x":0.0,"y":0.0,"description":"Main entrance"}
]
```

### 2.2 edge.json
```json
[
  {"fromId":"GATE_E","toId":"LIB","weight":220.0,"oneWay":false,"forbidden":false,"roadType":"MAIN_ROAD"}
]
```

### 2.3 user.json
```json
[
  {"username":"admin","passwordHash":"sha256$<64位十六进制哈希>","role":"ADMIN"}
]
```

## 3. 服务能力（PersistenceService）
- `loadGraph()` / `saveGraph(...)`
- `loadUsers()` / `saveUsers(...)`
- `backupData(name)` / `restoreData(name)`
- `loadGraphOrDefault()`：缺失或损坏时自动回退默认地图并重写文件
- `loadUsersOrDefault()`：缺失或损坏时自动回退默认用户并重写文件
