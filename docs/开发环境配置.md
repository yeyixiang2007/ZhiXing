# 开发环境配置说明

## 1. 基础环境
1. 操作系统：Windows/macOS/Linux 均可。  
2. JDK：建议 JDK 21（兼容 Java 8 语法）。  
3. IDE：VS Code（推荐安装 Java Extension Pack）或 IntelliJ IDEA。

## 2. 项目构建方式
1. Maven 方式（推荐）：项目已提供 `pom.xml`。  
2. 脚本方式（无需 Maven）：使用 `scripts/build.ps1` 与 `scripts/run.ps1`。

## 3. VS Code 配置
项目已提供：
1. `.vscode/settings.json`  
2. `.vscode/tasks.json`（可直接执行 Build/Run 任务）

## 4. 首次运行
```powershell
.\scripts\build.ps1
.\scripts\run.ps1
```
